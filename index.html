Here are some of my projects

  • Provincial Highways Pavement Condition Analysis (2014)

    RStudio, CSV

    In this project, we analyzed 2014 pavement condition data for Ontario's provincial highways using RStudio. The dataset included indices such as Distress Manifestation Index (DMI), wheel track rutting, and International Roughness Index (IRI), which were combined into a Pavement Condition Index (PCI). Using linear regression, random forest, and decision trees, we visualized and predicted pavement conditions to support better maintenance planning.
